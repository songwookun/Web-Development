# Web-Development

**안보라 국어 학원** 웹사이트 프로젝트입니다.
HTML, CSS, JavaScript만으로 구축한 SPA(Single Page Application)로, 별도 서버 없이 정적 호스팅만으로 운영할 수 있습니다.

> **Live Demo:** https://songwookun.github.io/Web-Development/

---

## 프로젝트 구조

```
Web/
├── index.html        # 메인 HTML (SPA 진입점)
├── css/
│   └── style.css     # 전체 스타일시트 (반응형 포함)
├── js/
│   ├── data.js       # 기본 데이터 및 localStorage CRUD 함수
│   ├── admin.js      # 관리자 인증 및 편집 기능
│   └── app.js        # SPA 라우터 및 페이지 렌더링
└── img/
    └── logo.png      # 학원 로고 이미지
```

## 주요 기능

| 기능 | 설명 |
|------|------|
| **SPA 라우팅** | Hash 기반(`#home`, `#instructors` 등) 라우팅으로 페이지 새로고침 없이 화면 전환 |
| **강사진 소개** | 강사 프로필 카드 (이름, 직책, 소개, 사진) |
| **교육과정** | 과정별 상세 정보 (대상, 수업 시간, 설명) |
| **입학 테스트** | 테스트 절차 안내 및 상세 정보 |
| **공지사항** | 목록/상세 보기, 검색 기능 지원 |
| **관리자 모드** | 비밀번호 인증 후 모든 콘텐츠를 인라인으로 추가/수정/삭제 가능 |
| **반응형 디자인** | 데스크탑/태블릿/모바일 모두 지원 (768px 이하 햄버거 메뉴) |
| **플로팅 전화 버튼** | 각 관(캠퍼스)별 전화번호를 빠르게 확인 및 연결 |

## 실행 방법

### 방법 1: 파일 직접 열기
별도의 설치나 빌드 과정이 필요 없습니다.

```bash
# 레포지토리 클론
git clone https://github.com/songwookun/Web-Development.git
cd Web-Development

# index.html을 브라우저에서 열기
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux
```

### 방법 2: 로컬 서버로 실행
```bash
# Python이 설치되어 있다면
python3 -m http.server 8080

# Node.js가 설치되어 있다면
npx serve .
```
이후 브라우저에서 `http://localhost:8080` 으로 접속합니다.

## 관리자 모드 사용법

1. 페이지 하단 푸터의 **저작권 문구**(© 2026 안보라...)를 **3회 빠르게 클릭**
2. 비밀번호 입력 창이 표시되면 관리자 비밀번호 입력
3. 상단에 초록색 **편집 모드** 바가 나타나면 활성화 완료
4. 각 콘텐츠 카드에 표시되는 **수정/삭제** 버튼으로 편집
5. **편집 종료** 또는 **로그아웃** 버튼으로 편집 모드 해제

> 관리자 비밀번호는 SHA-256 해시로 저장되어 있습니다.
> 변경하려면 `js/admin.js`의 `ADMIN_PW_HASH` 값을 교체하세요.
> 해시 생성 방법은 해당 파일 상단 주석을 참고하세요.

## 데이터 저장 방식

- 모든 데이터는 브라우저의 **localStorage**에 저장됩니다.
- 처음 접속 시 `js/data.js`에 정의된 기본 데이터가 자동으로 로드됩니다.
- 관리자가 수정한 내용은 해당 브라우저에만 저장되므로, **다른 기기나 브라우저에서는 기본 데이터**가 표시됩니다.
- 브라우저 캐시/데이터를 삭제하면 수정 내용이 초기화됩니다.

## 기술 스택

- **HTML5** - 시맨틱 마크업
- **CSS3** - CSS 변수, Flexbox, Grid, 반응형 미디어쿼리
- **Vanilla JavaScript** - 외부 라이브러리/프레임워크 없음
- **GitHub Pages** - 정적 사이트 호스팅

## 페이지 구성

| 해시 경로 | 페이지 |
|-----------|--------|
| `#home` | 메인 화면 (히어로 배너 + 각 섹션 미리보기) |
| `#instructors` | 강사진 소개 |

| `#notice/{id}` | 공지사항 상세 |

| `#terms` | 이용약관 |
| `#privacy` | 개인정보처리방침 |
