# Web-Development

**안보라 국어 학원** 웹사이트 프로젝트입니다.
HTML, CSS, JavaScript로 구축한 SPA(Single Page Application)이며, Supabase를 백엔드로 사용합니다.

> **Live Demo:** https://songwookun.github.io/Web-Development/

---

## 프로젝트 구조

```
Web/
├── index.html              # 메인 HTML (SPA 진입점)
├── css/
│   └── style.css           # 전체 스타일시트 (반응형 포함)
├── js/
│   ├── supabase-config.js  # Supabase 연결 설정
│   ├── data.js             # Supabase DB CRUD 함수
│   ├── admin.js            # 관리자 인증 및 편집 기능
│   └── app.js              # SPA 라우터 및 페이지 렌더링
└── img/
    └── logo.png            # 학원 로고 이미지
```

## 주요 기능

| 기능 | 설명 |
|------|------|
| **SPA 라우팅** | Hash 기반(`#home`, `#instructors` 등) 라우팅으로 페이지 새로고침 없이 화면 전환 |
| **Supabase DB 연동** | 모든 데이터를 클라우드 DB에 저장 — 관리자가 수정하면 모든 사용자에게 반영 |
| **관리자 인증** | Supabase Auth 기반 이메일/비밀번호 로그인, 비밀번호 변경 지원 |
| **RLS 보안** | 읽기는 누구나, 쓰기/수정/삭제는 인증된 관리자만 가능 |
| **배너 슬라이더** | 복수 배너 추가 가능, 2개 이상이면 5초 간격 자동 슬라이드 |
| **강사진 소개** | 강사 프로필 카드 (이름, 직책, 소개, 사진, 외부 소개 링크) |
| **교육과정** | 과정별 상세 정보 (대상, 수업 시간, 설명, 링크) |
| **입학 테스트** | 테스트 절차 안내 및 상세 정보 |
| **공지사항** | 목록/상세 보기, 검색 기능 지원 |
| **반응형 디자인** | 데스크탑/태블릿/모바일 모두 지원 |
| **플로팅 연락처 버튼** | 전화번호, 카카오톡, 네이버 카페 등 다양한 연락 수단 지원 |

## 관리자 모드 사용법

1. 지정된 방법으로 관리자 모드 진입
2. 이메일/비밀번호 입력 후 로그인
3. 상단에 초록색 **편집 모드** 바가 나타나면 활성화 완료
4. 각 콘텐츠의 **수정/삭제** 버튼으로 편집
5. **편집 종료** 또는 **로그아웃** 버튼으로 편집 모드 해제

편집 모드 바에서 **비밀번호 변경**도 가능합니다.

## 이미지 등록 방법 (Google Drive 활용)

### 1단계: Google Drive에 이미지 업로드

### 2단계: 공유 설정 변경
1. 이미지 파일(또는 폴더)을 **우클릭** → **공유**
2. **"링크가 있는 모든 사용자"** 로 변경

### 3단계: 링크 변환
복사된 링크에서 **파일 ID**를 추출하여 변환합니다:

| 구분 | 값 |
|------|-----|
| **복사한 링크** | `https://drive.google.com/file/d/파일ID/view?usp=sharing` |
| **사용할 URL** | `https://lh3.googleusercontent.com/d/파일ID` |

### 4단계: URL 입력
변환된 URL을 편집 모드에서 이미지 URL 필드에 붙여넣습니다.

## 데이터 저장 방식

모든 데이터는 **Supabase 클라우드 DB**에 저장됩니다.
- 관리자가 수정한 내용은 즉시 DB에 반영
- 누구든 사이트에 접속하면 DB에서 최신 데이터를 조회
- 30초간 클라이언트 캐시 적용 (불필요한 API 호출 방지)

## 기술 스택

- **HTML5** - 시맨틱 마크업
- **CSS3** - CSS 변수, Flexbox, Grid, 반응형 미디어쿼리
- **Vanilla JavaScript** - 외부 프레임워크 없음
- **Supabase** - PostgreSQL DB, Auth, RLS 보안
- **GitHub Pages** - 정적 사이트 호스팅

## 페이지 구성

| 해시 경로 | 페이지 |
|-----------|--------|
| `#home` | 메인 화면 (배너 슬라이더 + 각 섹션 미리보기) |
| `#instructors` | 강사진 소개 |
| `#curriculum` | 교육과정 목록 |
| `#test` | 입학 테스트 안내 |
| `#notices` | 공지사항 목록 (검색 가능) |
| `#notice/{id}` | 공지사항 상세 |
| `#map` | 오시는 길 (캠퍼스별 주소, 카카오맵 연동) |
| `#terms` | 이용약관 |
| `#privacy` | 개인정보처리방침 |
